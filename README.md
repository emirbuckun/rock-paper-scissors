## Run

Clone the project

```bash

git clone https://github.com/emirbuckun/rock-paper-scissors.git

```

Go to the project directory

```bash

cd rock-paper-scissors

```

Install dependencies

```bash

npm install

```

Start the server

```bash

npm run dev

```
